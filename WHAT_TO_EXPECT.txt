┌──────────────────────────────────────────────────────────────────────────────┐
│  WHAT HAPPENS IN THE BACKGROUND                                              │
└──────────────────────────────────────────────────────────────────────────────┘

When you click "Save Space Details":

1. JavaScript collects form data
   ↓
2. Sends to space_save.php
   ↓
3. PHP saves to database:
   - spaces table (plot size, orientation, etc.)
   - rooms table (room names and zones)
   - activity_log (user action)
   ↓
4. Returns space_id
   ↓
5. JavaScript calls vastu_analysis.php
   ↓
6. PHP analyzes space:
   - Calculates Fire element (SE, Kitchen)
   - Calculates Water element (NE, Bathroom)
   - Calculates Earth element (SW, Bedroom)
   - Calculates Air element (NW, Living Room)
   - Calculates Space element (Center)
   - Averages all 5 = Vastu Score
   - Generates recommendations
   ↓
7. Saves to analysis_results table
   ↓
8. Returns analysis_id
   ↓
9. JavaScript redirects to results page
   ↓
10. Results page calls api_get_analysis_results.php
   ↓
11. PHP fetches analysis data
   ↓
12. Returns JSON
   ↓
13. JavaScript displays beautiful visualization
